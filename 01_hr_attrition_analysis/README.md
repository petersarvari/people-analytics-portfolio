# HR Attrition Analysis

## Objective

Identify factors associated with employee attrition and create a management dashboard.

## Dataset

IBM HR Employee Attrition Dataset

## Tools

- PostgreSQL
- R
- Power BI

## Research Questions

1. What is the overall attrition rate?
2. Which departments have the highest attrition?
3. Does overtime increase attrition risk?
4. Is income associated with attrition?
5. Which employee groups are at highest risk?
